Hisaab Kitaab 
Hisaab Kitaab is a simple personal finance dashboard. It's designed to help students and young professionals see all their spending—from UPI apps like GPay to shopping on Amazon—in one single place.

What it does:

Expense Tracker: Shows exactly where your money goes (Food, Shopping, etc.) using clean charts.

Portfolio View: A quick look at your savings, FDs, and stocks (like Upstox or Zerodha).

Smart Suggestions: The app gives simple tips, like suggesting you save more when you've spent too much on Zomato or Swiggy.

Filters: You can pick specific dates to see how much you spent on a certain day or week.

Built With:

Python – The main logic.

Streamlit – For the clean, white dashboard interface.

Pandas – To organize and filter the transaction data.

How to run it locally:

Clone the folder:
git clone https://github.com/YOUR_USERNAME/Hisaab-Kitaab.git

Go into the folder:
cd "Hisaab Kitaab"

Install the tools:
pip install streamlit pandas plotly

Start the app:
streamlit run app.py

Why I built this:

As a student, tracking expenses across five different payment apps is a headache.
I wanted to build a "single screen" solution that tells me my current bank balance and warns me if I'm spending too much too fast.
